Nr. Request Methods Endpoints Response
1 POST /users New posted user
2 PUT /users/:id Updated user name
3 POST /users/:id/:movieTitle Add to favorites movie list
4 DELETE /users/:id/:movieTitle Deleted from favorites list
5 DELETE /users/:id Shows list without this deleted user
6 GET /movies Shows list of all movies
7 GET /movies/genre/:genreName Shows the genre object
8 GET /movies/directors/:directorName Shows the director's name and bio
